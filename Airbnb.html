This was my first Python project, where I aimed to learn Python EDA (Exploratory Data Analysis) and basic machine learning techniques through a small project. I chose a dataset on Kaggle for my research, focusing on identifying the main factors influencing Airbnb rental prices in New York City. The primary goal was to determine the appropriate machine learning prediction model for forecasting housing prices.

Analysis Results


  • There is no significant correlation observed among the variables in the dataset. However, Airbnb prices may potentially correlate with factors such as demand, location, and property type
  • The Random Forest model performs better in predicting housing prices on this dataset
